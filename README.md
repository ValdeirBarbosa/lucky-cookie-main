# lucky-cookie
# Curso Explorer 07
# Desafio estágio 05 explorer  
Esta aplicação exibe uma nova mensagem a cada click no biscoito da sorte, sortidas em um range de 50 mensagens
Otimo estudo para consilidar o aprendizado de CSS e Javascript
